# Facturador AFIP (PyQt6)

App de escritorio (PyQt6) inspirada en `desktop/files`, pensada para usuarios no técnicos.

## Ejecutar

Desde la raíz del repo:

```bash
uv run desktop/facturador_pyqt6_app/main.py
```

## Archivos generados

La app usa:

- Configuración: `~/.facturador/.env` (a través de `ARC_ENV_FILE`)
- Certificados: `~/.facturador/certs/`
